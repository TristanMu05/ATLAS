# Modes Documentation

## There will be 4 main modes for the system:

### 1. **IDLE Mode**: The system is powered on but not actively performing any operations. It waits for user input or scheduled tasks to transition to other modes.

### 2. **NORMAL Mode**: The system is performing its primary functions, such as data collection, processing, or communication. This mode is where the main operations occur.

### 3. **SAFE Mode**: The system is in a low power operation mode, prioritizing remaining online. This mode would be in case of voltage, temperature, or other critical system parameters being out of range. In this mode, the system would reduce power consumption and limit functionality to essential operations to protect itself.

### 4. **DIAGNOSTIC Mode**: The system is performing self-tests, diagnostics, or maintenance tasks. This mode may be used for troubleshooting, firmware updates, or other non-standard operations that require special handling.
